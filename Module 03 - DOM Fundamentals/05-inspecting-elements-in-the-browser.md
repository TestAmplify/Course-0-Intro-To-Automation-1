# 📝 Text Mode (Full Explanation)

### **How to Inspect Elements**

Right-click any element → **Inspect**

This highlights the element in the Elements panel.

---

### **What You Should Look For**

* Tag name (`button`, `input`, `div`)
* ID, class, name attributes
* Element hierarchy
* Whether the element is visible
* Whether the element is disabled

---

### **Why Inspecting Matters**

Inspecting helps automation engineers:

* Find stable locators
* Avoid dynamic attributes
* Understand nested structures
* Identify hidden elements
