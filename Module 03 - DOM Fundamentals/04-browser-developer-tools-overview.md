# 📝 Text Mode (Full Explanation)

### **What Are Browser Developer Tools?**

DevTools are built-in tools that let you:

* Inspect HTML
* Inspect CSS
* View network calls
* Debug JavaScript
* Analyze performance

Automation engineers rely heavily on DevTools.

---

### **Key Panels for Automation**

* **Elements** → Inspect DOM
* **Console** → View JavaScript logs
* **Network** → Track API calls
* **Application** → Cookies & Storage
* **Performance** → Rendering details

---

### **How to Open DevTools**

**Mac:**

```
Command + Option + I
```

**Windows:**

```
Ctrl + Shift + I
```
