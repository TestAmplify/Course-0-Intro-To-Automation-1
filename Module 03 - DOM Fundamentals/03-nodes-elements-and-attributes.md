---

# 📝 Text Mode (Full Explanation)

### **What Is a Node?**

A node is **any** item in the DOM tree.

Nodes include:

* Elements
* Text
* Comments
* Document nodes

---

### **What Is an Element?**

An element is a specific type of node — created from HTML tags.

Examples:

```
<button>Login</button>
<input type="email">
<div class="container"></div>
```

Automation interacts primarily with **elements**.

---

### **What Are Attributes?**

Attributes provide extra information about an element:

* `id="login-btn"`
* `class="form-input"`
* `type="email"`
* `placeholder="Enter name"`

Selectors often rely on attributes.

---

### **Why This Matters**

Good automation engineers know:

* Which attributes are stable
* Which are dynamic
* Which should be avoided

Attributes = the key to good locators.
