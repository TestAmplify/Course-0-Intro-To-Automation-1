# 📝 Text Mode (Full Explanation)

### **What Are CSS Selectors?**

CSS selectors identify HTML elements using:

* Tag
* ID
* Class
* Attributes
* Hierarchy

Automation tools can use these also.

---

### **Common CSS Selectors**

**By Tag:**

```
button
input
div
```

**By ID:**

```
#login-btn
```

**By Class:**

```
.button-primary
.form-input
```

**By Attribute:**

```
input[type="email"]
button[data-testid="submit"]
```

**By Hierarchy:**

```
div.card button
```

---

### **Why CSS Selectors Matter**

They are:

✔ Fast
✔ Easy to learn
✔ Supported everywhere
✔ Predictable

CSS is usually preferred over XPath for stability.
