# 🎯 Lesson Objective

* How to view HTML test reports
* How to read failures
* How to open trace files
* How to visually inspect run history

---

# 📝 Text Mode (Full Explanation)

### **Generate a Report**

After running tests:

```
npx playwright show-report
```

Opens an HTML report showing:

* Passed tests
* Failed tests
* Retry attempts
* Duration
* Attachments (screenshots, traces)

---

### **Screenshots & Traces**

Playwright can capture:

* Screenshots
* Videos
* Traces

Configured in your `playwright.config.js`.
