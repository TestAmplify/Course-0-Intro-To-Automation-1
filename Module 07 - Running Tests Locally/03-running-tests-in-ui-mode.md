# 🎯 Lesson Objective

* How to inspect DOM snapshots
* How to re-run tests visually

---

# 📝 Text Mode (Full Explanation)

### **Open UI Mode**

```
npx playwright test --ui
```

UI Mode lets you:

* Run tests visually
* Inspect each step
* View DOM at each action
* Replay failures
* Debug interactively

---

### **Why UI Mode Helps**

✔ Easier debugging
✔ Visual clarity
✔ Step-by-step timeline
✔ Instant re-runs

This is one of Playwright’s most powerful features.
