# 📝 Text Mode (Full Explanation)

### **What Is a Flaky Test?**

A flaky test:

* Sometimes passes
* Sometimes fails
* Without code changes

Very dangerous in automation.

---

### **Causes of Flaky Tests**

* Dynamic DOM
* Bad locators
* Manual waits
* Animations
* Slow network
* Timing issues

---

### **How to Avoid Flaky Tests**

✔ Use Playwright locators
✔ Use auto-waits
✔ Use proper assertions
✔ Avoid nth-child selectors
✔ Avoid manual waits
✔ Use stable attributes
