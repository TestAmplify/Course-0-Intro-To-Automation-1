# 📝 Text Mode (Full Explanation)

### **What Is Visual Studio Code (VS Code)?**

VS Code is a **lightweight, powerful, and free** code editor built by Microsoft.

Automation engineers use it because:

* It’s extremely fast
* It has excellent TypeScript support
* It integrates perfectly with Playwright
* It includes built-in Git tools
* It has a powerful debugging system
* Extensions add superpowers

VS Code is used by developers, SDETs, DevOps, and automation engineers worldwide.

---

### **Why Automation Engineers Use VS Code**

VS Code gives you:

✔ Syntax highlighting
✔ Auto-complete (IntelliSense)
✔ Integrated terminal
✔ Git/GitHub integration
✔ Debugging tools
✔ Playwright Test Runner extension
✔ Easy environment setup

Everything you need to build, run, debug, and organize automated tests.

---

### **How to Install VS Code**

Follow these steps:

1. Go to the official download page:
   [https://code.visualstudio.com/](https://code.visualstudio.com/)

2. Click **Download for Windows**, **Download for macOS**, or **Linux** (auto-detected).

3. Run the installer and follow the prompts:

   * Next
   * Accept license
   * Install
   * Launch VS Code

4. After installation, open VS Code and explore the interface briefly.

---

### **Install Essential Extensions**

Inside VS Code:

1. Click the **Extensions** icon (left sidebar).
2. Search and install the following:

**Required Extensions:**

* **Playwright Test**
* **ESLint**
* **Prettier — Code Formatter**
* **GitHub Pull Requests**
* **JavaScript and TypeScript Tools (built-in)**

**Optional but helpful:**

* **Material Icon Theme**
* **DotENV**
* **Error Lens**

These extensions make writing automation tests smoother, cleaner, and faster.

---

### **Open the Built-In Terminal**

You will use VS Code's integrated terminal to run Playwright commands.

Open it using:

**Windows/Linux:**

```
Ctrl + `
```

**Mac:**

```
Control + `
```

This is where you’ll run commands like:

```
npx playwright test
```

---

### **Checklist After Installation**

Make sure:

✔ VS Code opens correctly
✔ Terminal works
✔ Extensions installed
✔ You can create a folder and open it
✔ You understand where to write code and where to run commands

You're now ready to create your **first automation project folder** in Lesson 4.
