---

# 📝 Text Mode (Full Explanation)

### **HTML — Structure**

HTML creates the **skeleton** of a webpage.

Example:

```
<h1>Welcome</h1>
<button>Login</button>
```

Playwright interacts with HTML elements via locators.

---

### **CSS — Styling**

CSS controls:

* Colors
* Spacing
* Layout
* Animations

Automation does **not** interact with CSS directly, but CSS affects visibility and layout.

---

### **JavaScript — Behavior**

JavaScript handles:

* Click actions
* Input validation
* Popups
* Dynamic content

Automation must wait for JavaScript to finish loading before interacting.

---

### **Why This Matters**

Automation engineers often need to know:

* Why elements are not visible
* Why elements move or change
* Why dynamic content loads late

Understanding basic HTML/CSS/JS prevents confusion later.
