# 📝 Text Mode (Full Explanation)

### **What Is Playwright?**

Playwright is a modern automation framework built by Microsoft.
It allows you to automate:

* Chrome
* Firefox
* WebKit (Safari engine)

Using fast, reliable, modern APIs.

---
### **Playwright Docs ->> https://playwright.dev/docs/intro**
---

### **Why Automation Engineers Prefer Playwright**

✔ Auto-waiting
✔ Cross-browser testing
✔ More stable than Selenium
✔ Smart locators
✔ Fast parallel tests
✔ Built-in test runner
✔ Debugging tools included
✔ Excellent TypeScript support

Playwright is built for the **modern web**, especially apps built with:

* React
* Vue
* Angular
* Next.js
* And other SPA frameworks

---

### **Playwright’s Test Runner**

Playwright includes:

* Test runner
* Assertions
* Reporters
* Parallel execution
* Trace viewer
* Codegen tool

This means: **no additional libraries required.**
