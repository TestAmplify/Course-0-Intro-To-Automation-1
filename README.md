# ⭐ COURSE 0 — Intro to Automation Testing (Your First Automated Test)

Welcome to **Course 0** of the TestAmplify Test Automation Engineer Track — your official starting point in automation testing.

This course gives you **your first real automated test** using **Playwright + TypeScript**, while teaching you how modern web applications, browsers, DOM, locators, and debugging tools actually work.

Whether you are a **manual tester**, **beginner**, or **career-changer**, this course is designed to give you **immediate wins, real skills, and real confidence**.

---

# 🎯 Learning Outcomes

By the end of this course, you will:

✔ Understand how automated tests interact with real applications  
✔ Write and execute your **first automation script** using Playwright  
✔ Use **VS Code debugging tools** to diagnose test failures  
✔ Understand **DOM, selectors, browser interactions, and assertions**  
✔ Build a real mini-project: your **first automation smoke test**  
✔ Build confidence early so you can succeed in Course 1+
