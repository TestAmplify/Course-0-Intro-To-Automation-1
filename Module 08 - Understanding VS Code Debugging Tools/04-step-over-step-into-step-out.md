# 📝 Text Mode (Full Explanation)

### **Step Into**

Moves into the function call.

Useful when:

* You need to see inside a helper function
* You want to inspect parameters

---

### **Step Over**

Runs the current line and moves to the next.

Useful when:

* You don’t need to see inside internal Playwright functions

---

### **Step Out**

Exits the current function.

Useful when:

* You want to return to the parent flow

---

### **Why Stepping Matters**

Stepping helps you analyze:

* Execution flow
* Timing issues
* Unexpected page behavior
* Broken locators
